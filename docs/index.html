AI Assist Lab

기존 음성 비서보다 더 열려 있는 모바일 AI 에이전트 실험.

Concept

채팅으로 휴대폰 도구를 실행하는 AI 비서

앱 실행, 파일 열기, 문서 생성, 예약 작업을 하나의 대화 흐름에서 처리한다.

Difference

닫힌 비서가 아니라 확장 가능한 에이전트

  • 도구 호출 구조가 코드로 열려 있음
  • 로컬 모델과 API 모드를 선택 가능
  • 결과물을 직접 생성하고 실행까지 연결

Flow

입력에서 실행까지

사용자 요청
AI / 규칙 플래너
ToolCall
승인 또는 실행

Features

현재 가능한 일

  • 앱 실행과 앱 목록 조회
  • 웹 검색, 파일 열기, 캘린더/이메일 작성
  • PPTX/PDF 문서 생성
  • 일회성/반복/매일 예약 실행

Why It Matters

사용자가 직접 조립할 수 있는 비서

Siri/Bixby가 완성형 제품이라면, 이 프로젝트는 기능을 계속 붙일 수 있는 실험 플랫폼에 가깝다.

Limitations

아직은 MVP

  • 음성/OS 통합은 상용 비서보다 약함
  • 접근성 자동화는 권한과 안정성 검증 필요
  • 일부 한국어 문자열 인코딩 정리 필요

Tech Stack

  • tools도구 실행 라우터
  • documentsPPTX/PDF 생성
  • plan예약 저장과 실행

Summary
